# Product Scope

`35gateway` 是一个可自托管的 AI gateway。个人版提供最小控制台，商业版保留完整运营后台。

## Community Edition

- 多模型统一接入入口
- 用户控制台
- API Key 管理
- provider account 管理
- 模型目录与价格参考
- 请求日志与异步任务查看
- 设置与基础账户信息

## Commercial Edition

- 多模型统一接入入口
- 用户控制台
- 运营后台
- 用户与支付管理
- 模型运营与供应商运营
- 兑换码与活动发放
- 请求日志与异常追踪

## Not Included

- Creator 产品
- 工作流画布
- 多区域分布式控制平面

## Primary Users

- 独立开发者
- 小团队工程师
- 代理商或交付型技术负责人
- 想自己保留 key、路由和部署控制权的团队

## Default Flow

1. 登录控制台
2. 查看模型目录
3. 获取系统默认 key 或创建自定义 key
4. 用 key 调接口
5. 回看日志和任务

## Promise

- 单机可跑
- 自托管优先
- 多模型统一入口
- 价格、日志、任务状态可见

商业版另外提供：

- 运营后台
- 支付与用户运营
- 供应商与模型运营能力
- 商业授权与答疑支持
